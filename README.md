# 自動化排班表系統
本系統為複數店加盟主彈性分配人力資源提供輔助，能夠更快速安排班表雛型。旨在於透過員工自行填寫上班時段及地點意願（優先度），自動安排合理的工作時間及地點

## 架構
### 層級範例
- 組織A
    - 門市A
    - 門市B
    - 門市C

### 權限
權限賦予身份組
- 所有權限
- 手動安排員工班表
- 檢視所有門市班表
- 檢視所在門市(群組)班表
- 檢視個人班表
- 修改個人排班時段
- 修改個人地點偏好
- 修改所有人排班時段
- 修改所有人地點偏好
- ...

### 身份組
每個使用者可以擁有多個身份組
- 管理者
- 店經理
- FT
- PT
- ...(自由新增) 

### 門市
可以設定有特定身份組及權限的使用者才能編輯，檢視...

### 班表架構
- 精確到每 1 小時
- 時段為 7 天 * 24 小時
- 每小時都必須能夠填寫需求人數
- 需求人數可能會隨著商圈活動有所變化
- 以一週為一個排班周期
- 每一個地方都有獨立的班表

### 管理界面
- 組織的管理員只能夠管理自己組織下的門市
- 可以手動從員工清單上拖曳到表上
- 手動安排能夠覆蓋掉自動安排的結果
- 能夠檢視過去的班表
- 能過用顏色檢視人力需求熱力圖

### 員工界面
- 能夠填寫基本個人資料
    1. 姓名
    2. 聯絡電話
- 能夠填寫每一個地方的上班意願（比重和為1.0）
    1. 用UI控制（見圖例img001）
- 每週必須填寫可以排班的時段
    1. 使用UI選擇 （見圖例img002）
    2. 可以上下捲動
    3. 可以框選快速選取（在觸控設備上長壓後拖曳）
    4. UI自適應設備 
    5. 可以將當週的排班時段延續到下一週，當作是預設時段
